Min length for "Firstname" and "Lastname" is 3 characters, max is 30 characters. You should get a ConstraintViolationException if these conditions fail.

You should see the client side logging in your JS-Console. If you start the App with Eclipse you will also see the logging (Server and Client) on your Eclipse Console.

Console logging is single-lined!!! for the client. This is done with SingleLineTextLogFormatter.
